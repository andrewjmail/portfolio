Mortgage Calculator

A web app allowing users to input mortgage terms and see the mortgage debt burn down. They can also input overpayments to see how that effects the total amount paid.

Built with Vue.js as an experiment with the framework and uses the Element Ui component library.

There is also a chart.js component to display the breakdown of interest versus debt repayment for each year of the mortgage term.

Local Weather

A Page displaying the weather for the users current location. Uses Geolocation to determine the location, the forecast.io API to retrieve the weather and uses this information to generate hourly and 5 day forecasts with JavaScript. Users can also search for a location to see the weather anywhere in the world.

The app uses Media Queries to achieve a Responsive experience.
